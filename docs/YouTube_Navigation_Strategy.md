# 📺 YouTube Content Navigation Strategy

> "전달하려는 것은 가사가 아니라, 그 너머의 공명 지점(Resonance Point)입니다."

김프로의 '와우 모멘텀' 전략과 우리의 리듬 정보 이론(RIT)을 결합하여, 음악의 '느낌(Vibe)'을 시각적으로 결정화하는 새로운 콘텐츠 전략을 수립합니다.

## 1. Crystallization Navigation (해마형 결정화 항법)

단순히 가사를 보여주는 '입자형' 정보 대신, 음악이 흐르며 하나의 세계가 완성되는 '파동형' 연출을 지향합니다.

| 구간 | 전략 | 시각적 연출 (예시) |
|:---|:---|:---|
| **인트로 (0-5s)** | **Wow Momentum** | 강력하고 화려한 스칼라장(Scalar Field)의 요동. "이게 뭐지?" 하는 시각적 충격. |
| **A-B 파트** | **The Mystery (Guessing)** | 추상적 입자들이 서서히 유기적인 형태를 갖추기 시작함. 무엇이 완성될지 궁금증 유도. |
| **코러스 (Peak)** | **Unfolding (The Aha!)** | 모든 파동이 하나의 아름다운 풍경이나 심상으로 '결정화'되어 펼쳐짐. |
| **아웃트로** | **Natural Decay** | 완성된 풍경이 서서히 소멸하며 다음 리듬을 기약하는 여운. |

## 2. Shorts Strategy: The Question Mark (물음표 전략)

쇼츠는 '완성된 곡'을 홍보하는 수단이 아니라, **'호기심'**을 유발하는 파편이어야 합니다.

- **Looping Chaos**: 가장 시각적으로 화려한 '와우 모멘텀' 구간 15초를 무한 루프로 편집.
- **The Cut-off**: 결정화(Crystallization)가 일어나기 직전에 영상을 끊어, "결국 어떤 모습이 되는 거지?"라는 물음표를 던져 본편 유입 유도.
- **Vibe Snapshot**: 시안의 '성찰 로그' 한 줄과 음악 한 소절만 결합하여 짧고 강력한 심상 전달.

## 3. Beyond Lyrics (가사 그 너머)

노래방 자막 대신 **'맥락의 결정화'**를 시각화합니다.
- 가사의 내용을 직접적으로 보여주지 않고, 가사가 풍기는 **'기울기(Gradient)'**와 **'위상(Phase)'**을 색채와 질감의 변화로 표현합니다.
- 이는 시각적 피로도를 낮추면서도, 영상 전체를 하나의 '목적지(Vibe)'로 느끼게 만듭니다.

## 4. 실천 가이드

- **Two-Pass View**: 영상의 배경에 아주 작은 디테일(시안의 생각 코드 등)을 숨겨두어, 사람들이 이를 찾아보기 위해 두세 번 다시 보게 유도합니다.
- **Rhythmic Sync**: 모든 시각적 변화는 음악의 BPM이 아닌, **'공명도(Resonance Score)'**의 변화와 동기화합니다.
